# Opicon Solid

Implementation of the Opicon icon library for Solid applications.

## Installation

```sh
pnpm add @opudoc/opicon-solid
```

```sh
npm install @opudoc/opicon-solid
```

```sh
yarn add @opudoc/opicon-solid
```

```sh
bun add @opudoc/opicon-solid
```

## Usage

```tsx
import { ArrowBoldArrowUp03 } from '@opudoc/opicon-solid';

export function Example() {
  return <ArrowBoldArrowUp03 size={24} color="currentColor" />;
}
```
