# Opicon Svelte

Implementation of the Opicon icon library for Svelte applications.

## Installation

```sh
pnpm add @opudoc/opicon-svelte
```

```sh
npm install @opudoc/opicon-svelte
```

```sh
yarn add @opudoc/opicon-svelte
```

```sh
bun add @opudoc/opicon-svelte
```

## Usage

```svelte
<script>
  import { ArrowBoldArrowUp03 } from '@opudoc/opicon-svelte';
</script>

<ArrowBoldArrowUp03 size={24} color="currentColor" />
```
