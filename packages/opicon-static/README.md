# Opicon Static

Static SVG files and metadata for the Opicon icon library.

## Installation

```sh
pnpm add @opudoc/opicon-static
```

```sh
npm install @opudoc/opicon-static
```

```sh
yarn add @opudoc/opicon-static
```

```sh
bun add @opudoc/opicon-static
```

## Usage

Import individual SVG files:

```js
import arrowUp from '@opudoc/opicon-static/icons/arrow-bold-arrow-up-03.svg';
```

Or use the manifest:

```js
import manifest from '@opudoc/opicon-static/manifest.json' with { type: 'json' };
```
