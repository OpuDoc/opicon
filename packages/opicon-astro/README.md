# Opicon Astro

Implementation of the Opicon icon library for Astro applications.

## Installation

```sh
pnpm add @opudoc/opicon-astro
```

```sh
npm install @opudoc/opicon-astro
```

```sh
yarn add @opudoc/opicon-astro
```

```sh
bun add @opudoc/opicon-astro
```

## Usage

```astro
---
import ArrowBoldArrowUp03 from '@opudoc/opicon-astro/icons/arrow-bold-arrow-up-03.astro';
---

<ArrowBoldArrowUp03 size={24} color="currentColor" />
```
