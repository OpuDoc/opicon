# Opicon React

Implementation of the Opicon icon library for React applications.

## Installation

```sh
pnpm add @opudoc/opicon-react
```

```sh
npm install @opudoc/opicon-react
```

```sh
yarn add @opudoc/opicon-react
```

```sh
bun add @opudoc/opicon-react
```

## Usage

```tsx
import { ArrowBoldArrowUp03 } from '@opudoc/opicon-react';

export function Example() {
  return <ArrowBoldArrowUp03 size={24} color="currentColor" />;
}
```
