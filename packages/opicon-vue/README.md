# Opicon Vue

Implementation of the Opicon icon library for Vue applications.

## Installation

```sh
pnpm add @opudoc/opicon-vue
```

```sh
npm install @opudoc/opicon-vue
```

```sh
yarn add @opudoc/opicon-vue
```

```sh
bun add @opudoc/opicon-vue
```

## Usage

```vue
<script setup>
import { ArrowBoldArrowUp03 } from '@opudoc/opicon-vue';
</script>

<template>
  <ArrowBoldArrowUp03 :size="24" color="currentColor" />
</template>
```
