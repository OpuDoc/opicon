# Opicon Preact

Implementation of the Opicon icon library for Preact applications.

## Installation

```sh
pnpm add @opudoc/opicon-preact
```

```sh
npm install @opudoc/opicon-preact
```

```sh
yarn add @opudoc/opicon-preact
```

```sh
bun add @opudoc/opicon-preact
```

## Usage

```tsx
import { ArrowBoldArrowUp03 } from '@opudoc/opicon-preact';

export function Example() {
  return <ArrowBoldArrowUp03 size={24} color="currentColor" />;
}
```
